# Keyboard Letters Showing Wrong? (Fix)

## Problem

Some keys show wrong symbols.
Example:

* `"` and `@` swapped
* `#` or `\` in wrong place

This happens because your PC is using **UK keyboard layout**.

---

## Fix

### 1. Open Settings

Settings → Time & Language → Language & Region

### 2. Open Language Options

Click **English (United Kingdom)** → click **⋯ (three dots)** → Language options

### 3. Add US Keyboard

Click **Add a keyboard**
Select **United States – QWERTY**

### 4. Switch Keyboard

Press:

```
Windows + Space
```

Select **ENG US**

---

## Best Option

Remove **United Kingdom keyboard** and keep only **United States QWERTY**.

---

Done ✅
Your keyboard will work normally now.
